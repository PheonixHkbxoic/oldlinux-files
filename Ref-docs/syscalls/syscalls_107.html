sysinfo

SYNOPSIS

int sysinfo(struct sysinfo *info);

PARAMETERS

info: [out] buffer where to store the info.

DESCRIPTION

Returns system information and statistics. (What a surprise!!!) The sysinfo structure has the following layout: 

struct sysinfo }
        long uptime;                    /* Seconds since boot */
        unsigned long loads[3];         /* 1, 5, and 15 minute load averages */
        unsigned long totalram;         /* Total usable main memory size */
        unsigned long freeram;          /* Available memory size */
        unsigned long sharedram;        /* Amount of shared memory */
        unsigned long bufferram;        /* Memory used by buffers */
        unsigned long totalswap;        /* Total swap space size */
        unsigned long freeswap;         /* swap space still available */
        unsigned short procs;           /* Number of current processes */
        char _f[22];                    /* Pads structure to 64 bytes */
};

RETURN VALUE

On success zero is returned. On error -1 is returned and errno is set to EFAULT.
