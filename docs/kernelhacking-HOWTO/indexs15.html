FAQ

If you would like to suggest a question (and hopefully an answer too) for this FAQ then send them to me with KH-HOWTO QUESTION in the subject.

  • Q: I want to get started doing something useful, which project should I help out on?

    A: The best thing to do is subscribe to the kernelnewbies mailing list and ask if anyone would like to take a newbie onto their project

  • Q: Can you set me some kernel programming assignments?

    A: Sorry, I simply don't have time to do this for everyone who asks. I hope to add some more examples followed by tasks/assignments to each section of the document in the near future.
