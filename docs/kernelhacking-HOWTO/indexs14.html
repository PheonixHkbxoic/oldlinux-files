mailing lists

  • Kernelnewbies, Consists of a web page, IRC channel (\#kernelnewbies on irc.openprojects.net) and a mailing list.

  • Linux-kernel, a high volume list, post to it only after all other options are exhausted. It certainly should not be your "first port of call". Please go down this check list before posting.

    1. Read the FAQ

    2. Search the archives **FIXME** replace with aimsgroup URL

    3. Read the latest kernel-traffic, to see if the problem you are experiencing is a known issue.

    4. Do your homework first (e.g. read appropriate man pages, guides and HOWTO documents)

    5. Read & follow this posting guide

    6. Polish your message before posting; keep it short but informative.
