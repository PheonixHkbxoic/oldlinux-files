D Linux BBS List

  

  Printed here is a list of bulletin board systems (BBS) which carry Linux software. Zane Healy (healyzh@holonet.net) maintains this list. If you know of or run a BBS which provides Linux software which isn't on this list, you should get in touch with him.

The Linux community is no longer an Internet-only society. In fact, it is now estimated that the majority of Linux users don't have Internet access. Therefore, it is especially important that BBSs continue to provide and support Linux to users worldwide.
