INT 21,2A - Get Date

AH = 2A

on return:

AL = day of the week (0=Sunday)

CX = year (1980-2099)

DH = month (1-12)

DL = day (1-31)

- retrieves system date based on the DOS maintained clock

- updates ~BIOS Data Area~ current date and date rollover flag

at location 40:70
