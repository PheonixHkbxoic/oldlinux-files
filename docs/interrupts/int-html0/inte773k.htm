INT 14,3 - Get Serial Port Status

AH = 03

DX = zero based serial port number (0-1) (0-3 for AT)

on return:

AH = port status

AL = modem status

- for status bits see ~INT 14,STATUS~

- the status check performs a poll of the port and does not

perform character I/O
