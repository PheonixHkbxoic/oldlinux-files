INT 67,44 - Map Logical Page Into Physical Page Window (LIM EMS)

AH = 44h

AL = physical page number (0-3)

BX = logical page number (0 to total allocated minus 1)

DX = EMM handle

on return:

AH = 00 success

= error code (see ~EMS STATUS~)

- registers not listed are preserved
