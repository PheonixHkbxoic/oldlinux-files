INT 13,19 - Park Fixed Disk Heads (AT & newer)

AH = 19h

DL = drive number (0=A:, 1=2nd floppy, 80h=drive 0, 81h=drive 1)

on return:

AH = status (see ~INT 13,STATUS~)

CF = 0 if successful

= 1 if error

- available only on AT, XT 283 and PS/2 machines
