INT 21,9 - Print String

AH = 09

DS:DX = pointer to string ending in "$"

returns nothing

- outputs character string to STDOUT up to "$"

- backspace is treated as non-destructive

- if ~Ctrl-Break~ is detected, ~INT 23~ is executed
