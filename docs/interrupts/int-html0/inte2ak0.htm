INT 16,10 - Extended Wait for Keypress and Read Character (AT+)

AH = 10h

on return:

AH = ~scan code~

AL = ASCII character or zero if special function key

- available on AT and PS/2 machines with extended keyboard support

- similar to ~INT 16,0~
