INT 33,22 - Set Language for Messages

AX = 22h

BX = language number (with /L switch value):

= 0 English n/a

= 1 French F

= 2 Dutch NL

= 3 German D

= 4 Swedish S

= 5 Finnish SF

= 6 Spanish E

= 7 Portuguese P

= 8 Italian I

returns nothing

- only works with international version of the mouse driver

- see ~INT 33,23~
