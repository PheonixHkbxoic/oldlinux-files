Borland C++ DPMILOAD.EXE - GET ADDRESS OF ??? BY NAME

AX = FB42h
BX = 0005h
CX = selector of DPMILOAD data (see #03130)
DS:DX -> ASCIZ or CR-terminated name of ??? (case ignored)


Return:
DX = status (see #03129)
0000h successful
AX:BX -> ??? FAR function (called with two words on top of stk)
else
BX destroyed

See Also: AX=FB42h/BX=0006h - AX=FB42h/BX=000Eh
