IRQ8 - CMOS REAL-TIME CLOCK

Desc: This interrupt is called when the real-time clock chip generates an alarm or periodic interrupt, among others (see CMOS 0Bh). The periodic interrupt by default occurs 1024 times per second.

Notes: Many BIOSes turn off the periodic interrupt in the INT 70h handler unless in an event wait (see INT 15/AH=83h,INT 15/AH=86h).. May be masked by setting bit 0 on I/O port A1h
