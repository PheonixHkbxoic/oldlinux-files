NetSoft DOS-NET v1.20+ - INSTALLATION CHECK

AX = 4147h ('AG')
DX = 0000h


Return:
DX = 4147h if installed
DS:SI -> configuration data (see #02554)

Program: DOS-NET is a shareware networking package by Albert Graham

Note: This call is supported by CLIENT.COM, SERVER.COM, ROUTER.COM, and NETDOS.COM

See Also: INT 65/DX=4147h - INT 65/DX=4741h
