PRINDIR v9.0 - INSTALLATION CHECK

AX = 7F00h


Return:
AL = FFh if installed
BX = version (BH = major, BL = minor)
CX:DX -> ASCIZ signature "PRINDIR"

Range:
AH=7Fh is the default, may be changed at installation time

Note: Prior versions of PRINDIR used INT 7C or INT 7A

See Also: AX=7F01h"PRINDIR" - AX=7F02h"PRINDIR" - AX=7F03h"PRINDIR"
