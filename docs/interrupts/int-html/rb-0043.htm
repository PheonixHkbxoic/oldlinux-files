IRQ0 - SYSTEM TIMER

Desc: Generated 18.2 times per second by channel 0 of the 8254 system timer, this interrupt is used to keep the time-of-day clock updated

Notes: Programs which need to be invoked regularly should use INT 1C unless they need to reprogram the timer while still keeping the time-of-day clock running at the proper rate. Default handler is at F000h:FEA5h in IBM PC and 100%-compatible BIOSes. May be masked by setting bit 0 on I/O port 21h
