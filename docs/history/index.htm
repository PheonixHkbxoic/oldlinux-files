There is a complete set of kernels from kernel 0.99.14 to date. However, some kernels prior to 0.99.14 known to have been released appear to no longer exist. Such kernels have an entry in the relevant lists with just a version number thereon.

PLEASE NOTE: All kernels in these lists are referenced in bzip2 format where that is available, in gzip format where that is available and bzip2 format is not, and in the original UNIX compress format where neither is available. Please ensure that you have the relevant tool available on your system before downloading.

If you have any queries, please contact the maintainer of this page.

  4. Lists by CD-ROM volume: Linux Kernel Version Archives:

    • LKVA-001: Volume 1 contains all kernels released from the original release on 17th September 1991 through to the 2.0-pre14 release on 6th June 1996 that are known to still exist. This includes the entire collection of available 0.* kernels, and the entire 1.0, 1.1, 1.2, 1.3 and 2.0-pre series. Please note that several kernels prior to the kernel 0.99.14 release appear to no longer exist.

    • LKVA-002: Volume 2 contains all kernels released from 7th June 1996 to 31st December 1997 inclusive. This includes the 2.0 kernel series up to and including the 2.0.33 release and the 2.1 kernel series up to and including the 2.1.76 release.

    • LKVA-003: Volume 3 contains all kernels released from 1st January 1998 to 22nd January 1999 inclusive. This includes the 2.0 kernel series from the 2.0.34 release to the 2.0.36 release inclusive, the 2.1 kernel series from the 2.1.77 release to the 2.1.132 release inclusive, and the 2.2-Pre kernel series from the 2.2-Pre1 release to the 2.2-Pre9 release inclusive.

    • LKVA-004: Volume 4 contains all kernels released from 23rd January 1999 to 7th January 2000 inclusive. This includes the 2.0 series from the 2.0.37 release to the 2.0.38 release inclusive, the 2.2 series from the 2.2.0 release to the 2.2.14 release inclusive, and the 2.3 series from the 2.3.0 release to the 2.3.37 release inclusive.

    • LKVA-005: Volume 5 contains all kernels released from 8th January 2000 to 31st December 2000 inclusive. This includes the 2.2 series from the 2.2.15 release to the 2.2.18 release, the 2.3 series from the 2.3.38 release to the 2.3.51 release, the 2.4-pre series from the 2.3.99-pre1 release to the 2.3.99-pre9 release, the 2.4-test series from the 2.4.0-test1 release to the 2.4.0-test12 release and the 2.4.0-prerelease kernel release.

    • LKVA-006: Volume 6 contains all kernels released from 1st January 2001 to 9th March 2002 inclusive. This includes the 2.0.39 kernel from the 2.0 series, the 2.2 series from the 2.2.19 release to the 2.2.20 release, the 2.4 series from the 2.4.0 release to the 2.4.18 release and the 2.5 series from the 2.5.0 release to the 2.5.6 release.

    • LKVA-007: Volume 7 will contain all kernels released from 10th March 2002 onwards until it is full. At this stage, no such kernels exist.
