The Linux Bootdisk HOWTO

Tom Fawcett and Graham Chapman

v2.0, 25 June 1996

This document describes how to create Linux boot, boot/root and utility maintenance disks. These disks could be used as rescue disks or to test new kernels. Note: if you haven't read the Linux FAQ and related documents such as the Linux Installation HOWTO and the Linux Install Guide, then you should not be trying to build boot diskettes.
