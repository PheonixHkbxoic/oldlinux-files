Invariants

Here we define a set of invariant propositions. These propositions can be momentarily false during file manipulations in the ext2 file system manager. However, file invariants should be always be true for the set of files not currently manipulated by the file system manager. File system invariants should always be true when the file system manager is not currently manipulating the file system.
