time

SYNOPSIS

time_t time(time_t *t);

PARAMETERS

t: [out] where to store the time.

DESCRIPTION

Returns the time in seconds from 00:00:00 GMT January 1, 1970. If t is not NULL, the value is also stored there.

RETURN VALUE

The time (see description).
