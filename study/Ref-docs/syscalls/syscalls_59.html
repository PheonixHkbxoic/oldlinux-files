template

SYNOPSIS

pid_t fork(void);

pid_t vfork(void);

DESCRIPTION

Create a child task from the current task. The new task is an exact copy of the parent appart from the PID. vfork is simply an alias for fork.

RETURN VALUE

On success, the call returns the PID of the new task in the parent and returns 1 in the child. On error, returns -1 and sets errno to the following value: EAGAIN meaning that there is not enough memory to complete the call.
